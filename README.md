# 🎮 Simon Says Game

A fun and interactive Simon Says Game built using **HTML, CSS, and JavaScript**.

## 🚀 Features

- Random color sequence generation
- Dynamic level progression
- User input validation
- Flash animations
- Game Over effect
- Highest score tracking
- Responsive and attractive UI

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (ES6)

## 📂 Project Structure

```
Simon-Says-Game/
│
├── simon_game.html
├── simon.css
├── simon.js
└── README.md
```

## 🎯 How to Play

1. Press any key to start the game.
2. Watch the highlighted color carefully.
3. Repeat the sequence by clicking the buttons.
4. With every level, a new color is added.
5. The game ends when you click the wrong button.
6. Try to beat your highest score!
